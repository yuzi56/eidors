det3

PURPOSE

DET3 Parallel determinant for 3x3 matrices

SYNOPSIS

function D = det3(a)

DESCRIPTION 

DET3  Parallel determinant for 3x3 matrices
  D = DET3(A) calculates the determinant for each 3 rows of A. A must be 
  3N x 3. D is N x 1, such that
    D(i) = det( A( 3*(i-1)+(1:3) , 1:3 ) )
 but it is calculated faster.
