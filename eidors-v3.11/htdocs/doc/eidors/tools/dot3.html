dot3

PURPOSE

CROSS3 3D cross parallel cross product

SYNOPSIS

function d = dot3(a,b)

DESCRIPTION 

CROSS3  3D cross parallel cross product
  D = DOT3(A,B) calculates the dot product between rows of matrices A
  and B, both of which must be Nx3. D is Nx1 such that 
    C(i) = dot(A(i,:),B(i,:))
 but it is calculated faster.
